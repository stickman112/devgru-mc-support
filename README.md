# DEVGRU MC - Support

This is the official support ticket tracker for the DEVGRU MC gaming community.

Use this to report game server issues, get general help, report bugs, or suggest features.

---

## How to Open a Ticket

1. Click **Issues** at the top of this page
2. Click **New Issue**
3. Pick the template that matches your situation (see below)
4. Fill out the form and click **Submit new issue**

That's it. An admin will respond in the issue thread.

---

## Ticket Types

| Template | Use it when... |
|---|---|
| **Game Server Issue** | The server is lagging, crashing, kicking you, or won't let you connect |
| **General Help** | You need help with your role, permissions, account access, or anything else |
| **Bug Report** | Something on the server or site is broken or behaving wrong |
| **Feature Request** | You have an idea for something new or an improvement |

---

## What Happens After You Submit

1. **You'll get an automatic reply** confirming we received your ticket
2. **An admin will review it** and move it through our queue
3. **All updates happen in the issue thread** - GitHub will email you when someone replies (if you have notifications on)
4. **Your ticket gets closed** once it's resolved, with a note on what was done

---

## The Backlog

If your ticket gets labeled **backlog**, it means:
- We've seen it
- It's a valid request
- We just haven't gotten to it yet

You don't need to reopen or resubmit. We'll get to it. Feel free to leave a comment if your situation changes or becomes more urgent.

---

## Need Help Faster?

For urgent issues or real-time help, ping an admin directly in our **Discord server**.

A GitHub account is required to open a ticket here. If you don't have one, reach out on Discord and an admin can open one on your behalf.
